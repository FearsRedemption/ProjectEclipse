Archived rejected player and creature sprite PNGs from the pre-redo art pass.

These files are kept outside `Assets` so Unity does not import duplicate old art.
They were rejected because the sheets had visible white/matte artifacts and the
animations read as duplicated still images, squash/stretch, or recolors rather
than real step, attack, hurt, and death motion.

The active replacement files stay at the original paths under:

- `Assets/ProjectEclipse/Art/Player`
- `Assets/ProjectEclipse/Art/Creatures`
